# Project Rules

- **HANDOFF.md Maintenance:** You MUST keep the `HANDOFF.md` file in the root directory up to date at all times. Whenever you implement new features, change architectural logic, fix significant bugs, or release new versions (App or Firmware), you must automatically update `HANDOFF.md` to reflect these changes before concluding your task. Do not ask for permission to do this, simply treat it as part of your standard definition of done.
